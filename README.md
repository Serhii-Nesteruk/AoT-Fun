### AoT Game(Attack on Titan) 
AoT Game is a strategy game inspired by the Attack on Titan anime.
It is built on a simple, hands-on custom engine written with OpenGL.

Planned Features:
- City economies
- Trades between cities
- Country expansion system
- Combat systems (guns, security organizations, battles)
- Resource farming
- 2D and 3D modes
- UI(ImGUI)

### Dependencies:
- OpenGL 3.3
- GLFW 3.4.0
- glad
- glm

On Linux (Wayland):
On Wayland systems window decorations require *libdecor*:
```
sudo apt install libdecor-0-0 libdecor-0-plugin-1-gtk
```
